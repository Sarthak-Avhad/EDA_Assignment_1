Project Overview

This project analyzes a dataset of 10,000 used cars to explore the factors that influence car prices. The dataset includes car attributes such as brand, model, manufacturing year, engine size, fuel type, transmission, mileage, and ownership history.

The primary goal is to perform exploratory data analysis (EDA) and derive meaningful insights about the automobile resale market. The project can also be extended into a machine learning model for predicting car prices based on key features.
